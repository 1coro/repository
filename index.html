Instalace

Nejprve je třeba nainstalovat doplněk repozitáře, poté budete moci instalovat další doplňky

  1. Pro instalaci repozitáře stáhněte ZIP soubor repozitáže
  2. Spusťte Kodi/XBMC, a nainstalujte repozitár pomocí volby Nastavení -> Doplňky -> Instalovat doplněk ze zip archivu.

Pokud si nevíte rady, postupujte podle tohoto nebo tohoto video návodu.

Pro vývojáře

Jak přidat doplněk do tohoto repozitáře?

  • Pro Váš doplněk založte GIT repozitář - (github, google-code nebo bitbucket.org) nebo otevřte issue v kodi-czsk/repository a vytvoříme repozitář pod organizací kodi-czsk. Doplněk musí být v GIT repozitáři právě jeden a musí mít addon.xml v kořeni GIT repozitáře jako např. Befun.cz
  • Fork kodi-czsk/repository a vytvořte pull request, který přidá Váš doplněk mezi submoduly do master větve (použijte 
    git submodule add ...
    ) a zároveň přidejte doplněk do addons.py mezi automaticky releasované doplňky nebo otevřte issue s odkazem na GIT repozitář Vašeho doplňku
  • Jamile se Váš doplněk stane součástí repozitáře, pokračuje jeho vývoj ve vašem GITu v master větvi. Při změne verze v souboru addon.xml je pak doplněk automaticky vypublikován uživatelům.
